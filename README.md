# Job Application Tracker

## Overview

Job Application Tracker is a web-based dashboard application developed using HTML, CSS, JavaScript, and Chart.js. It helps users organize and manage job applications efficiently while tracking interviews, offers, and application status through an interactive dashboard.

The application stores data using browser Local Storage, allowing users to retain their information even after refreshing the page.

## Features

* Add new job applications
* Edit existing applications
* Delete applications
* Search applications by company name
* Filter applications by status
* Dashboard statistics overview
* Interview tracking section
* Offer tracking section
* Interactive Pie Chart visualization
* Interactive Bar Chart visualization
* Dark mode support
* Local Storage based data persistence
* Clear all stored data functionality

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Chart.js
* Browser Local Storage

## Project Structure

├── index.html
├── style.css

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Run `index.html` in any modern web browser.
